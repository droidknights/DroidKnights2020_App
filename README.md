# 👨‍💻👩‍💻DroidKnights2020 App 🛠🔧🔨
드로이드나이츠는 기술 세션으로 개발자들의 경험을 공유하는 안드로이드 컨퍼런스 입니다.
~2020년에는 4월 6일 코엑스 아셈볼룸 2층에서 진행될 예정입니다.~
2020년에는 온라인으로 행사를 진행하게 되었습니다!
**❤️9월 5일 LIVE 키노트로 돌아옵니다❤️**
